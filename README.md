📌 Project Management System

Welcome to Project Management System – your all-in-one solution for streamlined task management, team collaboration, and project tracking! 🚀

🌟 Features

✅ Task Management – Create, assign, and track tasks effortlessly.
✅ Team Collaboration – Communicate and work together in real-time.
✅ Project Tracking – Monitor progress with intuitive dashboards.
✅ Deadline Reminders – Stay on schedule with automated alerts.
✅ User Roles & Permissions – Control access and responsibilities.
✅ Cloud-Based – Access your projects anytime, anywhere.

📂 Tech Stack

🔹 Frontend: React.js, Tailwind CSS
🔹 Backend: Node.js, Express.js
🔹 Database: MongoDB
🔹 Authentication: JWT
🔹 Hosting: Vercel / Heroku

🚀 Getting Started

Follow these steps to run the project locally:

1️⃣ Clone the Repository

git clone https://github.com/jermanus74/project-management-system.git

2️⃣ Navigate to the Project Directory

cd project-management-system

3️⃣ Install Dependencies

npm install

4️⃣ Run the Development Server

npm run dev

5️⃣ Open in Browser
Visit http://localhost:3000 to explore the app!
